Reset Delivery Order

With this module you can reset your order to its default state and only inventory manager have the specific rights to use this module’s functionality. Additionally, this module will work on the sales order which are created after installing this module and will be disabled for the delivery orders if their flow follows Pick, Pack and Ship method.

Prerequisites before Installation

No prerequisites required before installation.

Configurations

No configurations needed.

Step-By-Step Guidelines

At first you need to create a sale order and on confirming the sale order a delivery order will be generated for that particular sale order. As shown in the below image, now it is in available state and if user will click on the “Cancel” button as per the default Odoo flow,delivery order will be canceled.

For the sale orders created after the installation of this module which are cancelled will get an option of “Set to Draft”. With the help of Set to Draft you can reset the order status to the default from cancelled.

Now as you can see in the below image, when clicked on “Set to Draft” the order status will change to default from Cancelled and the information of order will remain as it is.
